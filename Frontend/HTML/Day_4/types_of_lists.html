Html Lists

There are three main types of lists in HTML: unordered lists, ordered lists, and description (definition) lists.

1.Ordered List

  1. Used for lists where order does matter (such as steps or rankings).
  2. Items are numbered by default.
  3. Can be customized with different numbering styles (e.g., Roman numerals, letters).
  4. Uses the <ol> tag for the container and <li> for items.

2.Unordered List

  • Used for lists where order does not matter.
  • Items are typically marked with bullets (Disk) by default.
  • Can be customized with different bullet styles (e.g., circles, squares).
  • Uses the <ul> tag for the container and <li> for items.

3.Definition/Description list:

Used for terms and their descriptions.

description List :
A description list in HTML is used to display a list of terms and their corresponding descriptions or definitions. It is particularly useful for presenting name-value pairs, such as a dictionary-style format or questions and answers.
  • <dl> → Tag for creation of Description list
  • <dd> → Tag for Description definition
  • <dt> → Tag for Description term
