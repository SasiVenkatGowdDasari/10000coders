HTML

HTML (HyperText Markup Language) is the standard language used to create and structure web pages on the internet. It defines the content and layout of a webpage using a system of tags and elements.

HTML5 was introduced to replace older HTML versions (like HTML4 and XHTML.) and to make web pages more powerful, faster, and interactive without needing extra plugins like Flash.

HTML 5 Features:

  • Sematic elements
  • Multimedia Support
  • Form input enhancements
  • Geolocation API
  • Canvas & SVG
  • Storage(local & session)

Html Tags

Headings    :- Used to display titles and subtitles ranging from largest <h1> to smallest <h6>.
Paragarph   :- Represents a block of text separated from other content using the <p> tag
Anchor      :- Creates clickable links that navigate to other pages or sections using <a>.
Image       :- Displays pictures or graphics on a webpage using the <img> tag.
List        :- Organizes content into bullet points or numbered items with <ul> and <ol>.
Table       :- Presents data in rows and columns using <table>, <th>, <th>, and <td>.
Div         :- Groups block-level elements together for layout or styling purposes.
Span        :- Groups inline text or elements for styling within a line.
Form        :- Collects user input such as text, numbers, or selections.
Input       :- Accepts user data through various fields like text boxes, checkboxes, or buttons.
Link        :- Connects an external resource such as a CSS stylesheet to the HTML document.    
MultiMedia  :- Integrates audio and video content into webpages using <audio> and <video> tags,
