HTML Tags,Elements,Attributes

A tag is a keyword enclosed in angle brackets < > that tells the browser how to display content.

An element is everything from the opening tag to the closing tag, including the content inside.

An attribute provides additional information about an HTML element.Attributes are always written inside the opening tag and come as name-value pairs.

Basic Html Tags

Tag Description
<p> Defines a paragraph
<a> Defines a hyperlink
<img> Defines an image
<div> Defines a division or a section
<span> Defines a section in a document
<h1> to <h6> Defines HTML headings
<ul> Defines an unordered list
<ol> Defines an ordered list
<li> Defines a list item
<table> Defines a table
<marquee> Defines a scrolling text
<form> Defines an HTML form for user input
<mark> Defines marked/highlighted text
