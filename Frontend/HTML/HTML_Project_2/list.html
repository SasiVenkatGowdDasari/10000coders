Lists in HTML

Ordered List

An ordered list is a list of items that are presented in a specific order. Each item in the list is numbered, and the numbers indicate the order of the items.

To create an ordered list in HTML, you can use the <ol> tag. Each item in the list is defined using the <li> tag.

Example:

  1. Hii
  2. This is
  3. Ordered List

Unordered List

An unordered list is a list of items that are presented in no particular order. Each item in the list is marked with a bullet point.

To create an unordered list in HTML, you can use the <ul> tag. Each item in the list is defined using the <li> tag.

Example:

  • Apple
  • Banana
  • Mango
  • Cherry

Description List

A description list is a list of items that are defined by a term and a description. Each term is followed by its corresponding description.

To create a description list in HTML, you can use the <dl> tag. Each term is defined using the <dt> tag, and each description is defined using the <dd> tag.

Nested list

A nested list is a list that contains another list within it. The inner list can be either an ordered or unordered list.
